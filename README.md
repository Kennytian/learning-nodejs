#### JS 压缩与混淆
>JavaScript parser / mangler / compressor / beautifier toolkit http://lisperator.net/uglifyjs/

#### 安装：
> `yarn global add uglify-js` / `npm install -g uglify-js`

##### 用法：
最简单用法：
> `uglifyjs index.js -m -c -o index.min.js`
* -m 打乱命名
* -c 压缩文件
* -o 指定输出文件名

### learning node.js


#### Components:

`npm install -g supervisor`

supervisor 可以帮助你实现这个功能，它会监视你对代码的改动，并自动重启 Node.js。

`npm install mongoose`

Mongoose库简而言之就是在node环境中操作MongoDB数据库的一种便捷的封装，一种对象模型工具，类似ORM，Mongoose将数据库中的数据转换为JavaScript对象以供你在应用中使用。


#### ejs模板引擎语法
* <% %>： 用于js控制流，没有输出
* <%= %>： 输出转义的值到模板中
* <%- %>： 输出不转义的值到模板中
* <%# %>： 注释语句
* <%%： 输出’<%’的文本，相当于’<%’的转义形式
* <% -%>或<%= -%>或<%- -%>： 修剪掉换行符
